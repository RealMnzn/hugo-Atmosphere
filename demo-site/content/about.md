Hi, I'm the author of the [Atmosphere](https://github.com/realmnzn/hugo-Atmosphere) Hugo theme.

This is a dual-scene theme featuring **Sunny** (light) and **Rainy** (dark) modes with smooth atmospheric transitions. Toggle the sun/rain icon in the header to switch scenes.

## Features

- Scene system with smooth crossfade transitions
- PJAX-powered page navigation
- Full-text search
- Waline comment system integration
- Post encryption (AES-GCM)
- Image click-to-zoom
- Timeline, Interests, and Links page layouts
- Responsive and reading-first design
